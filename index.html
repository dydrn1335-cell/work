Google Apps Script를 무료 서버로 사용합니다.

  1. 구글 시트 새로 만들기
  2. 상단 메뉴 확장 프로그램 → Apps Script 클릭
  3. 기존 코드 전체 삭제 후 아래 코드 붙여넣기
  4. 💾 저장 → 배포 → 새 배포 클릭
  5. 유형: 웹 앱 / 액세스: 모든 사용자 → 배포
  6. 표시된 웹 앱 URL을 아래에 붙여넣기 → 저장
